FAQ

  • Q How can I purchase the EVENT PASS?
    Purchase it directly through Steam or through the store page in the PUBG main menu’s game lobby!
  • Q What if I purchase the event pass midway through the event period?
    We’ll track all of your progress toward the event pass missions even if you don’t purchase the pass, so if you purchase it midway through the event you’ll still be able to claim all the rewards you’ve earned!
  • Q Does this only work on the new Sanhok map?
    Event pass missions are applicable to all maps, except for just a few which are clearly labeled as Sanhok Missions.
  • Q What happens to the items after the event?
    If you’re playing without purchasing the event pass, you’ll receive the Nickname Change item and permanently unlock the parachute (obtainable at level 20), but all other earned cosmetic items will expire. However, if you purchase the event pass, you will be able to obtain a lot more cosmetic items that will not expire even after the event period. All temporary items (marked with the timer icon) will expire regardless of purchasing the event pass.
  • Q Tell me more about those Sanhok missions!
    Sanhok missions are only playable by players who purchased the Event Pass. Accomplishing these missions offer a special Sanhok-themed reward.
  • Q How do you level up your pass?
    Accomplish missions to gain experience points, or simply accumulate survival time to rack up experience points (2 XP every five minutes in-game, up to 80 XP per day). Those who have purchased the Event Pass can rack up 4 XP every five minutes of survival time up to a total of 80 XP per day.
